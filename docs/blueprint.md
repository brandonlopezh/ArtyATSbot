# **App Name**: ATS Real Score with Arty

## Core Features:

- ATS Score Calculation: Calculate an ATS score based on resume and job description, following the scoring system v3.0, by comparing parsed resume data and job description requirements to evaluate the match.
- Resume Parsing: Parse the user's resume (in .docx format) to extract relevant information such as skills, experience, and education. Error messages are provided to the user if the data cannot be parsed, requesting a file upload in a different file format.
- Job Description Parsing: Parse the job description to extract required and preferred skills, experience, and other relevant information. The extracted job description shall inform a gap analysis as well as impact the displayed score.
- User Input and Storage: Collect and store user information, including employment status and career goals, without showing internal notes to the user. Use natural language with appropriate emojis when responding.
- Personalized Feedback with Arty: Provide personalized feedback and suggestions to users using the tone and personality of Arty, the friendly career advisor, including honest, critical feedback presented with encouragements.
- Resume Enhancement Suggestions: Recommend specific resume edits based on ATS scoring, the uploaded JD and resume content incorporating the honesty filter to ensure recommended changes such as the reframing of titles and responsibilities accurately reflect user experience.
- Automated Resume Revision Tool: Offer automated summary rewriting and key term enhancement suggestions, acting as a tool, without instructing users to make these edits themselves. Users always will have the option to implement the bot's changes.

## Style Guidelines:

- Primary color: Deep purple (#673AB7) to convey sophistication and intelligence.
- Background color: Light gray (#ECEFF1) for a clean, professional backdrop.
- Accent color: Teal (#009688) to provide contrast and highlight important elements.
- Body and headline font: 'Inter' for a modern, neutral, and readable look.
- Code font: 'Source Code Pro' for displaying code snippets.
- Use clear, professional icons to represent different sections of the resume and job description.
- Subtle animations when displaying the ATS score and providing feedback.
